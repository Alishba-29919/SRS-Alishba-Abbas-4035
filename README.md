🍽️ Vocal-Dine – AI-Powered Voice-Interactive Smart Menu System
📌 Overview:

Vocal-Dine is an AI-powered web application designed to modernize restaurant ordering systems using voice interaction. It allows customers to browse menus, place orders using voice commands, and receive smart recommendations, while enabling restaurant admins to manage orders and view analytics in real-time.

🛠️ Tech Stack:
Frontend: React.js 

Backend: Flask   

Database: Firebase  

AI Engine: Web Speech API 


🍽️ Example Scenario:

Imagine a customer enters a restaurant.

Instead of asking for a menu, the customer scans a QR code placed on the table. As soon as the QR code is scanned, the menu opens on their mobile phone.

Now, instead of scrolling manually, the customer simply speaks:

“Show me burgers”

The system listens to the voice, converts it into text, and displays burger items on the screen.

Then the customer says:

“Order Zinger Burger”

The system processes this command, adds the item to the order, and shows a confirmation message.

At the same time, this order is automatically sent to the admin dashboard in the kitchen.

The restaurant staff can see the order in real-time, accept it, and start preparing the food.


🚀 Key Features
QR Code Integration: Every table has a unique QR code to access the digital menu instantly.  

Voice-to-Text Ordering: Customers can "speak" their order (e.g., "Add 2 Burgers and a Coke") instead of clicking.  

Real-time Admin Dashboard: Restaurant staff receives orders immediately through a live dashboard.  

AI Recommendations: The system suggests complementary items (like fries or drinks) to enhance the order.  

Order Summary: Users can review their cart and total bill before final confirmation.
